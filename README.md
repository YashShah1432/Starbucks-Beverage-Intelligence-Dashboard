# Starbucks Beverage Intelligence Dashboard ☕

## Project Overview

The Starbucks Beverage Intelligence Dashboard is an interactive Business Intelligence and Data Analytics solution developed in Power BI to analyze nutritional patterns, beverage characteristics, and category-level insights across Starbucks menu offerings.

This dashboard transforms raw beverage and store data into actionable insights through advanced visualizations, KPI tracking, and interactive filtering. The project focuses on analyzing calories, sugar content, caffeine levels, beverage preparation methods, and product distribution to support data-driven decision-making and nutritional awareness.

The dashboard interface and custom visual design elements were designed using **Figma** and integrated into Power BI to create a modern and professional user experience.

---

## Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 Power BI Desktop – Primary platform for dashboard development and reporting.
- 📂 Power Query – Data cleaning, transformation, and preparation.
- 🧠 DAX (Data Analysis Expressions) – Used for KPIs, calculated measures, and dynamic analysis.
- 🔗 Data Modeling – Built relationships and aggregations to support interactive reporting.
- 🎨 Figma – Designed the custom dashboard background and UI components.
- 📁 File Formats – `.pbix`, `.csv`, `.png`

---

## Data Source

**Source:** Public Starbucks Menu Nutritional Dataset

The analysis is based on two datasets:

- `Strabucks Beverage Nutrition Dataset.csv`
- `Starbucks Global Locations Dataset.csv`

The datasets contain beverage nutritional information, menu categories, preparation methods, caffeine levels, sugar content, protein values, and Starbucks global presence information.

## Features & Highlights

### Business Problem

Consumers increasingly seek healthier beverage options and nutritional transparency before making purchasing decisions.

However, comparing calories, sugar levels, caffeine content, and beverage preparation methods across multiple product categories can be difficult when working with raw data.

This dashboard converts raw Starbucks beverage data into meaningful business insights through interactive visual analytics.

### Goal of the Dashboard

The dashboard aims to:

- Analyze Starbucks beverage nutritional information.
- Compare beverage categories using calories, sugar, protein, and caffeine metrics.
- Identify beverages with the highest caffeine content.
- Explore category-wise beverage distribution.
- Enable interactive filtering based on beverage preparation methods and nutritional ranges.
- Deliver actionable insights through interactive visualizations.

Area chart displaying average calorie levels across beverage categories.

#### Starbucks Global Presence

Bar chart illustrating Starbucks presence across selected countries.

**Insight:** Canada and Japan demonstrate a strong Starbucks market presence among the displayed regions.

#### Beverage Category Distribution

Donut chart showing the distribution of beverages across menu categories.

**Insight:** Espresso-based beverages contribute a significant share of the Starbucks menu portfolio.

#### Average Caffeine by Category

Horizontal bar chart comparing average caffeine levels across beverage categories.

**Insight:** Brewed Coffee contains the highest average caffeine concentration among all beverage categories.

#### Top 5 Highest Caffeine Beverages

Highlights the beverages with the highest caffeine levels.

#### Interactive Filters

The dashboard includes:

- Protein Range Slicer
- Beverage Preparation Filter

These filters allow users to dynamically explore nutritional patterns and beverage characteristics.

## Business Insights

- Brewed Coffee contains the highest average caffeine concentration among all beverage categories.
- Smoothies and Frappuccino-based beverages exhibit the highest calorie levels.
- Sugar content varies significantly across beverage categories, highlighting differences in nutritional profiles.
- Espresso-based products dominate the menu distribution.
- Beverage preparation methods influence nutritional composition and caffeine content.
- Interactive filtering enables users to identify beverages aligned with specific dietary requirements.
